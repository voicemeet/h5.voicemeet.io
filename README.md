## 准备

> 需要 Node 版本 20+

## 使用

### 开发

只需要执行以下命令就可以在 <http://localhost:3000> 中看到

```bash
pnpm dev
```

### 构建

构建该应用只需要执行以下命令

```bash
pnpm build:pro
```

然后你会看到用于发布的 `dist` 文件夹被生成。
